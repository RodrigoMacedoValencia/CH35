Bootstrap-Intro

Clases principales y de uso en Bootstrap

class="container"

Ocupa las 12 columnos de grid que proporciona css, pero deja un margen a los costados

class="row"

inica que el elemento contenedor se comporta como una fila dentro de grid

class="col-n/xx"

Permite establece el numero de columnos que ocupe un elemento/componente. se puede delimitar tanto con numero y nommen claturas predefinidas ejemplos:sm, md, lg, xl, xxl.

class="container-fluid"

Ocupa el tamaño completo del viewport

Sticky footer

Pin a footer to the bottom of the viewport in desktop browsers with this custom HTML and CSS.

Use the sticky footer with a fixed navbar if need be, too.
